# CLI

## CLI COMMEND

```sh
$ touch a.txt				# a.txt를 생성
$ cd ~						# home으로 이동 Change Directory
$ mkdir test				# test폴더/디렉토리 생성. MaKe DIRectory
$ cd test					# test폴더로 이동
$ cd ..						# 상위 폴더로 이동
$ cd /						# root 디렉토리로 이동
$ cd -						# 뒤로 가기 이전 폴더로 이동
$ ls						# LiSt 현재
$ pwd						# 현재 내가 위치하고 있는 폴더. Present Working Directory
$ echo '테스트' > a.txt	  # 테스트 라는 글자를 a.txt에 넣음
$ echo '수정' > a.txt			#
$ rm a.txt					# a.txt 삭제
$ mv a.txt test/			# 테스트 폴더로 이동

```
